# CorpoDG Trip593 — Plataforma de Viajes

Sistema web para la agencia de viajes **CorpoDG Trip593**. Catálogo de destinos y paquetes turísticos, reservas de vuelos con Sabre GDS, pagos con Stripe, y asistente IA (Cory).

| Componente | Stack |
|------------|-------|
| Backend | Django 6.0 + DRF + PostgreSQL |
| Frontend | Vue 3 + Vite + Playwright |
